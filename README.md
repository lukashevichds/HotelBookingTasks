# HotelBookingTasks

Автоматизация решения задач онлайн бронирования и информационной поддержки клиентов с использованием технологий чат-ботов


ИС для пользователей, для бронирования отелей, с интеграцией чат-ботов и сопровождения клиентов.
Клиент может забронировать отель, в этом ему поможет чат-бот.

Разрабатываемое приложение предназначено для автоматизации сопровождения клиентов сервиса онлайн-бронирования отелей. Система должна обеспечивать не только учет бронирований, но и формирование чек-листа действий, связанных с каждым бронированием, а также предоставление пользователю справочной информации через чат-бот.


Наше приложение, а точнее макет WPF, показывает сам процесс сопровождения клиента.
